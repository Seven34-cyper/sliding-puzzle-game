# 🧩 Puzzle Master - O Enigma Prismático

Bem-vindo ao **Puzzle Master**, um jogo de puzzle deslizante premium, minimalista e altamente customizável. Desafie sua mente e organize fragmentos de ícones da cultura pop para revelar obras de arte completas.

## 🚀 Funcionalidades

- **Múltiplos Temas**: Escolha entre 5 temas visuais únicos:
  - 🎸 **AC/DC**: Estilo rock 'n' roll vibrante.
  - 🌑 **Shadow the Hedgehog**: Estilo escuro e impactante.
  - ⚡ **Flash Reverso (Zoom)**: Energia amarela e vermelha.
  - 🏍️ **Motoqueiro Fantasma**: Chamas e vingança.
  - 🤖 **Transformers**: Clássico Autobots.
- **Dificuldade Adaptativa**:
  - **Casual (3x3)**: Perfeito para relaxar.
  - **Mestre (4x4)**: O desafio clássico de 15 peças.
  - **Elite (6x6)**: 35 peças para quem tem foco.
  - **Lenda (8x8)**: 63 peças para os verdadeiros mestres.
- **Interface Premium**: Design baseado em Glassmorphism, com fundos dinâmicos e animações fluidas.
- **Dicas Inteligentes**: Botão "Revelar Números" que ajusta o tamanho da fonte com base no tabuleiro.

## 🎮 Como Jogar

1. Abra o arquivo `index.html` em qualquer navegador moderno.
2. Selecione seu **Tema** preferido no menu inicial.
3. Escolha o nível de **Dificuldade**.
4. Clique nas peças para movê-las para o espaço vazio.
5. O objetivo é organizar as peças na ordem numérica (1 até o fim) para formar a imagem completa.

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível.
- **CSS3**: Estilização avançada com variáveis, animações e efeitos de blur.
- **JavaScript (Vanilla)**: Lógica do jogo, manipulação de DOM e algoritmos de embaralhamento garantidamente solucionáveis.
